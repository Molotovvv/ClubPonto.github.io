A propos du Club

Le Club Informatique est un club au Lycee des Pontonniers. Il est dirigé par Mme.Moula et a été crée par Alex Sin et Lina Debouz. Ila été crée pour l'apprentissage des languages informatique et de la programmation de différente chose comme la robotique. Les horaires sont tout les mercredis de 12h-14h a midi (il faut ramener a manger).

Au Programme

Robotique

Le Club inclus aussi de la robotique. Pour les premières semainde nous allons coder un robot sur arduino.Le Robot que nous allons programmer aura des capteurs que nous pouvons controler avec arduino. Plus tard nous pourrons participer a une competition. La robotique permet de comprendre comment travaille les programmes et un peu le travail d'ingénieurs en robotique.

Informatique

Le Club inclus aussi de la programmation. Apres la Robotique nous pouvons passer sur de la proggramation avecde différente langue comme l'html ou le css qui sont les languages utiliser pour créer ce site. Vous etes surement familier avec le python qui est un language de code pour les algorithme principalement, mais nous allons explorer des languages pour le web design etc..
